# Luminous Salon - Professional Edition (Fortune)

## 概要 (Overview)
Luminous Salon Professional は、伝統的なタロット占術と現代のAI技術を融合させた、高度な占術鑑定ツールです。
無料版の視覚的体験に加え、APIを通じた「Luminous Reading（深層鑑定）」機能を搭載しています。

## 特徴 (Features)
- **Luminous Reading**: OpenAI/Gemini 等のAPIを活用し、展開されたタロットカードから相談内容に即した具体的かつ詩的な解釈を自動生成します。
- **マルチ・スプレッド対応**: ケルト十字法やヘキサグラムなど、専門的なタロットリーディングに対応した複数の展開法をサポート。
- **高精細アセット**: 1909年版タロット（ウェイト・スミス版）のデジタルリマスター画像を使用。

## セットアップ (Setup)
1. **APIキーの設定**:
   - `index.html` または設定画面を開き、使用するサービスのAPIキーを入力・保存してください。キーはブラウザのローカルストレージに安全に保管されます。
2. **画像アセットの配置**:
   - `assets/images/tarot/` フォルダ内にカード画像が配置されていることを確認してください。

## 使用方法 (Usage)
1. `index.html` をブラウザで開きます。
2. 占いたい内容（問い）を入力します。
3. タロットカードを展開し、「AI鑑定を実行」ボタンを押すと、カードの象徴に基づいた詳細な回答が表示されます。
